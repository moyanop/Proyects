### [Sistema de Gestión — Lubricentro Asensio](./asensio-lubricantes/)

Sistema integral de gestión para lubricentro con módulos de **inventario**, **punto de venta**, **facturación electrónica AFIP**, **historial de servicios vehiculares**, **reservas online** y **marketing predictivo** con recordatorios automáticos por WhatsApp.

Incluye gestión de roles (Admin, Operario, Mecánico, Cliente), cálculo de costo inverso para operarios, trazabilidad completa de movimientos de stock, generación de remitos PDF con QR y analíticas de ventas en tiempo real.

<summary>📌 Ver stack del proyecto</summary>

| **Backend**        | Node.js, Express, TypeScript, Prisma ORM, Zod, Swagger                                                    |
| **Base de Datos**  | PostgreSQL                                                                                                |

| **Notificaciones** | Resend (Email), node-cron (tareas programadas)                                                            |
| **DevOps**         | Docker (multi-stage builds), Docker Compose, Nginx                                                        |
| **Testing**        | Jest, Vitest, Testing Library, Supertest                                                                  |

</details>

### [Velo — Plataforma de Delivery](./velo-app/)

Marketplace de **delivery de comida** que conecta comercios, clientes y repartidores. Sistema completo con **3 aplicaciones móviles** (Cliente, Comercio, Repartidor), **panel web de administración**, comunicación en **tiempo real vía Socket.IO**, pagos integrados con **Mercado Pago** (OAuth + split payments) y tracking GPS de pedidos.

Publicada en **App Store** y **Google Play Store**. Operando en producción en **veloapp.com.ar** con 44 modelos de base de datos, 28 módulos de API y worker dedicado con tareas programadas.

<details>
<summary>📌 Ver stack del proyecto</summary>

<br>

| Capa                | Tecnologías                                                                                         |
| ------------------- | --------------------------------------------------------------------------------------------------- |
| **App Móvil (x3)**  | React Native 0.81, Expo SDK 54, TypeScript, NativeWind, React Navigation 7, Zustand, TanStack Query |
| **Panel Web**       | React 19, Vite 7, TypeScript, TailwindCSS, React Router 7, Recharts, Leaflet                        |
| **Backend**         | Python 3.11, FastAPI 0.115, SQLAlchemy 2.0, Alembic, Pydantic v2                                    |
| **Tiempo Real**     | Socket.IO (python-socketio), Redis 7+ (pub/sub + cache)                                             |
| **Base de Datos**   | PostgreSQL 15+                                                                                      |
| **Autenticación**   | JWT (access 30min + refresh 30 días), bcrypt, Google Sign-In                                        |
| **Pagos**           | Mercado Pago (OAuth, split payments, webhooks)                                                      |
| **Mapas**           | Google Maps API (Geocoding, Distance Matrix, Directions)                                            |
| **Notificaciones**  | Firebase Cloud Messaging (FCM)                                                                      |
| **Infraestructura** | VPS Ubuntu, Gunicorn + Nginx, systemd                                                               |
| **Testing**         | Pytest, HTTPX, Playwright (E2E), Locust (carga)                                                     |

</details>
